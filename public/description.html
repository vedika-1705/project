Project Description

Audio-to-Sign Language Translator

This project is designed to help individuals with hearing impairments by translating spoken words into sign language gestures. The system listens to audio, processes the speech, and displays the corresponding sign language animations in real-time.

Key Features

  • Real-time Speech Recognition – Converts spoken words into text.
  • Sign Language Display – Maps recognized words to sign language animations.
  • User-Friendly Interface – Simple and accessible UI for ease of use.
  • Customizable Dictionary – Expandable word mapping for better accuracy.

Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend (if needed): Node.js (Express)
  • Database (if needed): MongoDB
